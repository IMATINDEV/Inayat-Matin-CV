# Inayat Matin - Software Engineering CV 📄

## 📌 Overview
This repository contains my professional CV, custom-built from scratch using **HTML5** and **CSS3**. 

As I transition into software engineering, I wanted my CV itself to be a demonstration of my foundational technical skills, rather than relying on a pre-made word processor template. 

## 🚀 Live Version
[View the live web version of my CV here](https://imatindev.github.io/Inayat-Matin-CV/)

## 🛠️ Technologies Used
* **HTML5:** For semantic structuring and accessible content.
* **CSS3:** For styling, layout, and print-optimization.

## ✨ Key Features
* **Clean, Modern UI:** Designed for high readability and a professional aesthetic.
* **Print-Optimized:** Includes specific `@media print` CSS queries to automatically strip out browser headers/footers, adjust margins, and format perfectly onto an A4 PDF without breaking page elements.
* **ATS Friendly:** Text-based HTML structure ensures Applicant Tracking Systems can easily parse the content.

## 🖨️ How to Save as PDF
If you have downloaded this repository and want to generate the PDF version:
1. Open the `inayat_cv.html` file in any modern web browser (Chrome, Safari, Edge).
2. Press `Ctrl + P` (Windows) or `Cmd + P` (Mac) to open the print dialog.
3. Select **"Save as PDF"** as the destination.
4. *(Optional)* Ensure "Background graphics" is checked to keep the styling.
5. Click Save.
